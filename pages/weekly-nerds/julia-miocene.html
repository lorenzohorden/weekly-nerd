NAAM: Julia Miocene

DATUM: 17 / 03 / 2025

info over spreker:

Julia Miocene maakt animaties met HTML en CSS, ze heeft daar een youtube kanaal over en is best bekend online.

CSS animations

Julia heeft vooral animaties laten zien, dus ik heb weinig om hierover te vertellen. Een aantal dingen zijn me bijgebleven die ik belangrijk vond: Als je een animatie maakt met meerdere bewegende delen is het belangrijk om goed te nesten, zet bijvoorbeeld een onderarm in een arm en een hand in een onderarm. Op deze manier als je de arm draait draait alles wat eraan vast zit automatisch mee. Ook heeft ze verteld dat het animeren van de height property in 3D heel inefficient is voor performance en dat je het beter kunt doen met transform vanwege performance.

Mijn mening over de lezing:

Ik vond het super interessant wat Julia deed met die CSS animaties, alleen voor mezelf heeft het niet heel veel nut. Dat komt omdat ik meer naar de logische kant van het web wil en al het design wat ik ga maken word meer een UI layout dan een realistische animatie van een mug. Wel was die tip van het niet animeren van de height voor performance inzichtelijk, ik heb namelijk voor CSS een 3D kraan gemaakt en alles wat geanimeerd word word aangepast op de height, dat verklaart waarom mijn laptop 3 keer per dag leeg ging. Erg jammer dat ik dat nu pas te weten kom, want om dat op te lossen moet ik mijn hele cuboid base veranderen en daar heb ik geen tijd meer voor voor de deadline, wel handig voor een volgende keer.
