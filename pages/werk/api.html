VAK: API

wat vond ik van de les:

Dit vak viel het meest in mijn straatje, ik vind webservers opzetten leuk en werken met externe data is een leuke uitdaging. De vrijheid die we kregen bij het kiezen van een onderwerp was aan het begin niet fijn, omdat ik de scope van de opdracht niet helemaal begreep en mijn API kon een stuk minder dan die van tevoren zei. Dit forceerde me een beetje om eigen logica te schrijven voor dingen die ik anders via de API gedaan zou hebben wat een leuke uitdaging was.

wat vond ik van mijn resultaat:

De site die ik uiteindelijk gemaakt heb is super leuk geworden en ik vind het spelletje zelf heel leuk om te spelen, waardoor ik veel getest heb en bugs eruit heb kunnen halen. Ik ben tevreden over hoe ik de data ophaal en verwerk en wat ik eraan toe heb gevoegd.

conclusie:

Dit vak was voor mij niet heel leerzaam, omdat ik hier best goed in ben. Ik heb wel kennis van BT gebruikt om te communiceren tussen de server en de front-end. Dat maakte het wel makkelijker dan bij vorige vakken. Bij dit vak kwam ook weer het punt naar boven van meer voorbereiden voordat je begint te coderen, wat ik weer niet gedaan heb. Hierdoor kon op het einde mijn idee niet voldoen aan de eisen voor een voldoende, toen ben ik voor de herkansing terug gegaan naar het tekenbord en heb ik een super leuk idee bedacht wat ook leuk was om aan te werken.
