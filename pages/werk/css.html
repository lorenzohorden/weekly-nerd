VAK: CSS

wat vond ik van de les:

Ik denk dat CSS mijn minst sterke taal is van web, zeker toen ik aan dit vak begon. Gelukkig kregen we van verschillende docenten verschillende dingen te leren over alles wat met CSS te maken heeft. Het was ook fijn dat we ons in konden schrijven voor extra uileg over bepaalde onderwerpen.

wat vond ik van mijn resultaat:

Ik was van plan aan het begin om meerdere kamers te maken met heel veel verschillende dingen die je uit en aan kan zetten. Dat is totaal niet gelukt en daar ben ik vrij ontevreden over, maar ik vind het wel knap hoe snel ik 3D door had en hoe snel ik met variabelen bezig was. Het was vooral jammer dat ik achteraf erachter kwam dat ik dingen anders wilde doen, maar toen had ik er geen tijd meer voor.

conclusie:

Ik ben heel blij met de dingen die ik geleerd heb over 3D, dat was van tevoren een beetje mijn doel en die heb ik zeker gehaald. Ook heb ik in zijn algemeen veel geleerd over CSS, waarvan de belangrijkste nesting is. Ik heb het hier niet perfect toegepast, maar ik zie wel heel erg het nut ervan in en ga het 100% gebruiken om betere code te schrijven in de toekomst.
